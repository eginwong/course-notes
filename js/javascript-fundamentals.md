## Security
* use SSL
* use cookie attributes - Secure and HttpOnly
* Use HTTP Header: Strict-Transport-Security
* Cross-site Scripting Attacks - use CORS (`Access-Control-Allow-Origin`) or CSP (`Content-Security-Policy`)
